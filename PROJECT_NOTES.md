# Mock Interview Platform - Project Notes

## MVP Scope

### What's Included
- Next.js with TypeScript
- React components
- shadcn/ui component library
- Tailwind CSS for styling
- Lucide React icons

### What's NOT Included in MVP
- **No Database**: The MVP will use client-side state management only (React state, localStorage, etc.)
- **No Authentication**: No user login/signup functionality in the initial version
- Data will be stored locally in the browser and will not persist across devices

### Future Enhancements
When ready to scale beyond MVP:
- Add Supabase for database and authentication
- Implement user profiles and persistent data
- Add real-time matching and scheduling features
- Integrate video calling functionality
- Build feedback and rating system

## Tech Stack
- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
